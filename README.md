<h1>Para la práctica de desactivación del firewall de Windows 10 y 11, se necesita lo siguiente:</h1>
<h4>1. Que el antivirus tenga el modo "Protección contra alteraciones" desactivado.</h4>
<h4>2. Estos comandos fueron probados en un Digispark Attiny85, el cual fue configurado a través de Arduino.</h4>
<h4>3. Los comandos fueron diseñados para ser ejecutados en un teclado con el idioma en inglés; si está en otro idioma, puede que no funcionen.</h4>
